# garden-grove-mma

אפליקציית כרטיס אירוע חומ"ס — קריסת מיכל מתיל מתאקרילט (MMA) במפעל GKN Aerospace, גרדן גרוב, קליפורניה (מאי 2026).

חלק מפרויקט **60 שניות חומ"ס**.

## תוכן
- **מצב** — נתוני אירוע חיים, שעון BLEVE, שני תרחישי כשל.
- **מפה** — מפת Leaflet אינטראקטיבית: מוקד האירוע, רצפטורים ציבוריים (בתי ספר, פארקים, בי"ח, מקלטים), אזור פינוי מקורב וטבעות מרחק.
- **החומר** — זיהוי MMA, יהלום NFPA 704, תכונות פיזיקליות, השפעות בריאותיות.
- **מנגנון** — פילמור בורח, כשל המעכב, מנגנון החסימה, התערבויות שנשללו, פעולות המומחים, שיקולי כיבוי.
- **מידע** — ציר זמן ומקורות.

## טכני
- קובץ סטטי בודד (`index.html`), Leaflet via CDN, ממשק iOS, RTL עברית.
- נפרס אוטומטית ל-Vercel בדחיפה ל-main.
